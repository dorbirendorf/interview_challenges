# Surprise API
by Dor Birendorf

## how to run it locally

1. Install NodeJs 
2. Open terminal in the root of the project (or cloned repository)
3. Install dependencies using: 
```bash
npm install 
```
4. run 
```bash
node app.js 
```
and wait until terminal prints:
```bash
Server is up on port:3000 
```

5.Enjoy (:

## Usage 
navigate to : 
[http://localhost:3000/api/surprise?name=Ryan%20Gosling&birth_year=1980](http://localhost:3000/api/surprise?name=Ryan%20Gosling&birth_year=1980)

try to change the name and birth year manually to get different results.
